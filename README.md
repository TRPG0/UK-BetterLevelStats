# BetterLevelStats

Customize the level stats by toggling on or off everything from level name, to difficulty, to time, to major assists, to final rank achieved.

Install via [Thunderstore](https://thunderstore.io/c/ultrakill/p/TRPG/BetterLevelStats/), or by following the steps below:

1. Download and install [BepInEx 5.4.21](https://github.com/BepInEx/BepInEx/releases/tag/v5.4.21) in your ULTRAKILL root folder. *Do not use any pre-release versions of BepInEx 6.*

2. Start ULTRAKILL once so that BepInEx can create its required configuration files.

3. Install [PluginConfigurator](https://github.com/eternalUnion/UKPluginConfigurator).

4. Download `BetterLevelStats.zip` from the [Releases](https://github.com/TRPG0/BRC-NoTutorial/releases) page and extract the contents into `BepInEx\plugins`.
